class airflow.ti_deps.deps.base_ti_dep.BaseTIDep[source]¶

Bases: object

Abstract base class for dependencies that must be satisfied in order for task instances to run. For example, a task that can only run if a certain number of its upstream tasks succeed. This is an abstract class and must be subclassed to be used.

is_met(ti, session, dep_context=None)[source]¶

Returns whether or not this dependency is met for a given task instance. A dependency is considered met if all of the dependency statuses it reports are passing.
