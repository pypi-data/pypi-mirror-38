Welcome to jtypes.pyjnius
=========================

**jtypes.pyjnius** is a Python library for accessing Java classes.

  `PyPI record <https://pypi.python.org/pypi/jtypes.pyjnius>`__.

This documentation is divided into differents parts.
We recommend you to start with :ref:`README`, :ref:`installation`, and then
head over to the :ref:`quickstart`. You can also check :ref:`android` for
specific example for the Android platform. If you'd rather dive into
the internals of **jtypes.pyjnius**, check out the :ref:`api` documentation.
