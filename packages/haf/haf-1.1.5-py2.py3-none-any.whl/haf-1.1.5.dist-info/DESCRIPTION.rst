# haf

    The http api auto test framework. 

> based on pytest & allure
> xlsx easy cases

## How to get it

```shell
    pip install haf --upgrade
```

## How to run

### mkdir at testcases

```shell
    mkdir testcases
```

### create xlsx file 

### move file.xlsx to testcases

### run 

```shell
    python -m haf
```
