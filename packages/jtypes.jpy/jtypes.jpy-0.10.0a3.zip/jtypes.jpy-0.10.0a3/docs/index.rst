Welcome to jtypes.jpy's documentation!
======================================

**jtypes.jpy** is a *bi-directional* Java-Python bridge allowing you to call
Java from Python and Python from Java.

  `PyPI record <https://pypi.python.org/pypi/jtypes.jpy>`__.
